node.js মডিউল কি?

node.js এ মডিউল হচ্ছে একগুচ্ছ ফাংশন যা একটি ফাইল বা একাধিক ফাইল বা ফোল্ডারে থাকতে পারে। মডিউল ব্যবহারের অনেক সুযোগ সুবিধা রয়েছে। এর প্রধান সুবিধা হল মডিউলে থাকা ফাংশন বা ব্লক পুনরায় ব্যবহারযোগ্য। যার ফলে একটি মডিউল এর জন্য একবার কোড করেই একাধিকবার প্রোগ্রামে সেটি ব্যবহার করা যায়।

node.js এ তিন ধরনের মডিউল রয়েছে।

কোর মডিউলঃ কোর মডিউল বলতে node.js ইন্সটল করার সাথে সাথেই node.js এর সাথে বিল্ট-ইন যেসকল মডিউল থাকে সেগুলোকে বোঝায়। কোর মডিউল গুলো require() ফাংশনের সাহায্যে কোডের মধ্যে সরাসরি ব্যবহার করা যায়।

লোকার মডিউলঃ আমরা যদি আমাদের node.js প্রজেক্টে একটি ফাইল ক্রিয়েট করে সেখানে এক বা একাধিক ফাংশন তৈরি করি এবং পরবর্তীতে আমাদের প্রজেক্টে সেই ফাংশনগুলো রিকোয়ার করে ব্যবহার করি তাহলে সেটিকে বলা হবে লোকাল মডিউল।

থার্ড পার্টি মডিউলঃ node.js এর জন্য অনলাইনে অনেক মডিউল এভেলেবেল রয়েছে যেগুলো npm প্যাকেজ ম্যানাজার দিয়ে ইন্সটল করে ব্যবহার করতে হয়। এসব মডিউলকেই থার্ড পার্টি মডিউল বলা হয়। থার্ড পার্টি মডিউল গুলো ব্যবহার করার নিয়ম কোর মডিউল গুলোর মতই।
